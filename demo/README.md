# AEVA离线演示页面使用指南

## 📋 概述

这是一个**完全离线可用**的AEVA平台演示页面，专门为面试演示场景设计。即使在没有网络连接或API不可用的情况下，也能完整展示AEVA的所有核心功能。

## 🎯 设计目的

1. **应对网络故障**：面试现场网络不稳定或无法访问外网
2. **避免API依赖**：无需Claude API Key，使用预置Mock数据
3. **流畅演示体验**：加载速度快，操作响应迅速
4. **专业视觉效果**：现代化UI设计，突出技术专业性

## 🔧 技术细节

### 离线可用原理

1. **无外部依赖**
   - 所有CSS/JS内联
   - Chart.js使用CDN但有fallback
   - 即使Chart.js加载失败，页面仍可用

2. **Mock数据**
   - 所有数据硬编码在HTML中
   - 图表数据预置
   - 无需后端API

3. **纯前端实现**
   - 无需Web服务器
   - 直接用file://协议打开
   - 跨平台兼容
